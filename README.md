# Kubernetes__study

### 커맨드 설명
  - docker search A :  A에 대한 docker명령과 함께 명령을 사용하여 Docker Hub에서 사용 가능한 이미지를 검색
  - docker pull A: A 이미지를 컴퓨터에 다운로드
  - docker run -it A: A의 최신 이미지를 사용하여 컨테이너를 실행
